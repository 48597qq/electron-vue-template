# electron-vue-template

> 基于electron-vue的模板项目，解决了部分bug，并且升级了所有开发包到最新版本，方便以后自己开发使用。

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
